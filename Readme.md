## Readme

This repository contains the written *R* code used in the manuscript "Beyond Targets: Benefits of a Multi-Assessment Approach in Determining Chemical Water Quality for Water Reuse" by Specker et al. submitted to Water Research in June. 2026. The code was written by Jan Specker.

Please download the entire project folder (RProject) to run the individual scripts
successfully and ensure that respective *R* packages are installed. Scripts are
numbered in ascending order and have to be run in the corresponding order.
